VETERAN HUB — Uetersen
VETERAN HUB — это центральная точка доступа (линк-хаб) для сообщества ветеранов и диабетиков в городе Итерзен. Проект объединяет все необходимые инструменты для оперативной связи, контроля медицинских данных и взаимодействия с ботами-помощниками.

🎯 Цель проекта
Создание быстрого, удобного и надежного интерфейса для экстренной связи, обмена информацией и мониторинга состояния здоровья в рамках проекта Tech.chebur.shop.

🛠 Ключевые компоненты
SOS-ID Platform: Интеграция с платформой sos-id.online для доступа к медицинским QR-профилям.

Оперативная связь: Прямые переходы в Telegram-группы для экстренных ситуаций (SOS_Diabetes_Gruppe).

Автоматизация: Удобный доступ к ботам-помощникам (Бот Nick, Veteran Pionier Bot, Бот Ваня для расчета таможенных сборов).

Экосистема: Прямой переход в Teeladen (магазин чая).

Интерактивность: Блок для быстрой отправки запросов в групповой чат.

🖥 Технические особенности
Design: Минималистичный "военный" стиль (темная тема с акцентными цветами).

FontAwesome: Использование иконок для визуальной навигации.

Responsive: Адаптивная верстка, оптимизированная для использования на мобильных устройствах в критических ситуациях.

UI/UX: Интуитивное расположение кнопок с цветовым кодированием (красный для тревоги, синий для медицины, оранжевый для инструментов).

🚀 Установка и деплой
Хостинг: Файл index.html размещается в корневой директории вашего веб-сервера.

Ассеты: Убедитесь, что файл kommando.png (профиль) находится в той же директории, что и index.html.

Безопасность: Проект статичен и не требует баз данных на стороне клиента, что обеспечивает высокую скорость загрузки.
